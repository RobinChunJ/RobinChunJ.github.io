使用GitHub+hexo+icarus搭建个人博客

一、前言

csdn目前花里胡哨的东西太多，所以就想在更花里胡哨的地方——全网最大同性交友网站GitHub上搭建个人博客。所以头一篇当然是如何搭建。当然大部分都是copy过来的，也主要是为了熟悉MarkDown语法。

二、简介

Hexo是一个快速、简洁且高效的博客框架。使用MarkDown（或其他渲染引擎）解析文章，在几秒内可以利用炫酷的主题生成静态网页。Hexo官网

三、安排

1.Git安装

首先进入 Git-Downloads页面，无脑安装。

测试安装是否成功：

1、鼠标右击出现Git GUI Here和Git Bash Here选项。

或者

2、命令提示符输入git –version，如果对应有版本说明安装成功。

2.注册Github账号和建仓库

仓库名命名格式为: 自己的用户名+github.io

3.Node.js安装

下载 | Node.js 中文网，自行选择对应系统。
安装包下载完成之后双击下载，无脑下载，需注意的还是不要下载到了C盘。
确认Node.js安装成功：

cmd输入命令node -v，打印出版本号即可。

cmd输入命令npm -v，打印出版本号即可。

4.Hexo安装

建立一个Hexo文件夹, cd 进入Hexo文件夹
输入命令：

npm install -g hexo-cli

查看版本 hexo -v,验证HExo安装成功

4.1 初始化Hexo

输入hexo init myBlog,自动初始化

cd myBlog进入myBlog文件夹

npm install

like this

其中_config.yml为主配置文件

themes为主题文件夹，用以存放主题其中有个landscape文件夹为Hexo的默认主题，该文件下也有个_config.yml文件为主题配置文件

输入hexo -server或者hexo -s运行Hexo.Ctrl+c可停止

打开浏览器输入localhost:4000即可在本地预览博客

3.2部署

打开博客配置文件_config.yml进行配置

部署之前先生成静态文件hexo -generation或者hexo -g 运行后会在myBlog文件夹中生成public文件夹。

接着安装hexo-deloyer-git输入npm install hexo-deloyer-git --save

最后输入hexo -d或者hexo -deploy部署到刚刚建立的Github仓库中去

现在就可以在浏览器中输入用户名.github.io访问个人博客

4.更换主题

icarus 是一个三栏结构的自适应主题，预置了大量第三方插件，可以方便使用者快速建站。整体风格简约，适合各种类型的站点，无论是技术文章，还是摄影佳作，都可以完美承载。Hexo-theme-icarus

效果图

进入myBlog文件夹下，打开git bush下载主题

之后在themes文件夹里会有一个icarus文件

打开博客设置文件_config.yml设置theme为icarus以及一些其他设置，下面为个人的参考，

这里有主题官方的参考文档

试看效果

4.3写文章/显示文章主图

myBlog文件夹下打开git bash输入hexo new “文章名字”
